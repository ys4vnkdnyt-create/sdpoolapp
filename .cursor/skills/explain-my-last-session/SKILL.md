# Explain my last session

## When to use
- User asks what we covered last time
- User wants a recap before continuing
- User is catching up on phone or desktop

## Steps
1. Name the topic and which project area it sits in
2. Give a short summary of the latest work and how it fits the bigger project
3. Use plain English only
4. Write 1–4 sentences, then wait for their reply

## Do not
- Don’t dump the full chat log
- Don’t start new work unless they ask
- Don’t use jargon without a plain definition
- Don’t send more text until the user confirms they understand and got it
